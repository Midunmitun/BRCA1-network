# BRCA1 Protein Interaction Network

## What this does
Fetches live protein interaction data for BRCA1 from the STRING database 
and visualizes the network of proteins that interact with it.

## Why this matters
BRCA1 mutations disrupt an entire DNA repair system, not just one protein. 
This tool maps those connections to understand how a single gene change 
affects a whole biological system.

## How to run
1. Install dependencies: pip install requests networkx matplotlib
2. Run: python brca1_network.py
3. Output: brca1_network.png

## Data source
STRING Database - https://string-db.org
